Lean canvas

The Lean Canvas is a lean startup adaptation of the Business Model Canvas. The lean canvas focuses on addressing broad customer problems and solutions and delivering them to customer segments through a unique value proposition. You can read more about it on Wikipedia or The Lean Startup.

The whole idea

The whole idea of all that is to use simple tool online without spending money on buying existing solutions or printing lean canvas every time you decide to make changes.

How it works

You must have cookies enabled to save template that you are working on locally. I am still looking for saving them on server somewhere.
The canvas is represented as a set of columns. Each column consists of one or two directories, which has got its own name and description. Each directory is a folder for notes that you may create, and is supposed to organise them into different categories, such problems, solutions and etc. You can add note to a directory with + note
Note can be edited using Edit or deleted using Delete The new thing in this version is tags, so now you can assign multiple tags to the note using dropdown menu, instead of only one previously.
Create a tag with + tag button, select colour by clicking on the left square and type tag's name in. You cannot create tags from within editor (but it is something to think about). And there is no drag-n-drop feature yet.

Menu controls

You can visit github page and load previous version using controls on left side. New layout allows you to load new clear template. Save saves template locally using cookies, Load loads stored template. When you go to the website, it will try loading stored template anyway. Save as Gist and Load Gist save and load gist from Github respectivelly. It is recommended not only save your work locally, but also save it as Gist, thus later you can paste link and load template again.

Compatibility with older version

New lean canvas can load templates that were created using v.1.0.2, though tags (note colours) will not be imported.
