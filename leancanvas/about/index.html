Lean canvas

The Lean Canvas is a lean startup adaptation of the Business Model Canvas. The lean canvas focuses on addressing broad customer problems and solutions and delivering them to customer segments through a unique value proposition. You can read more about it on Wikipedia or The Lean Startup.

The whole idea

The whole idea of all that is to use simple tool online without spending money on buying existing solutions or printing lean canvas every time you decide to make changes.

How it works

The canvas is represented as a set of columns. Each column consists of one or two directories, which has got its own name and description. Each directory is a folder for notes that you may create, and is supposed to organise them into different categories, such problems, solutions and etc. Directory has a number of controls:
Note has several controls to work with it:
Top bar includes: New to load new clear canvas, everything that was created before will be deleted. Save to save current canvas to Github as Gist and save url into cookies.
Note that if cookies are disabled, nothing will be saved and error message will be displayed.
Save as Gist saves current canvas on Github as a Gist and shows you link when it is done (here one example of it https://gist.github.com/anonymous/b1c5d8ff55e199f53a22). Sometimes it is worth to copy and save link to continue working, in case cookies are disabled or cleared every day. Load Gist allows you to load saved Gist link and build canvas, that you may have been working on. You have to paste (or type if you want) link that you can get using Save as Gist.
Note that Github API has a limit for creating and loading anonymous Gists (about 60 requests per hour). If you exceed this you will get an error and will be unable to save data.
